# Thread 관리 기능
## Thread 생성
1. Thread 생성 버튼
   버튼 클릭시 AI Thread 생성
   THread 목록에 New Chat추가
   우측 패널에 해당 Thread 열림
   3초 이내 중복 생성 시 경고 표시
2. Thread 생성 +
	클릭 시 새 Thread 생성
	3초 이내 중복 생성 시 경고 표시
3. CH/DM 입력란 '/AI', '/먀' 명령어
	/AI 또는 /먀 명령어 선택 후 질의문 입력 시 Enter 키를 통해 새로운 Thread 생성 및 해당 질의문 요청
	/AI 대소문자 구분 없음
4. CH/DM 입력란에  @@ 명령어
   @@ 입력 시 새로운 Thread 생성 및 우측 패널에 해당 Thread 열림
   @@ 뒤에 질의문 입력 후 Enter 키를 통해 새로 생성된 Thread에 질의문 요청
   입력 예시
## Thread 삭제
1. Thread 삭제
	마우스 오버시 표시
	클릭 시 삭제 여부 확인 팝업 표시
2. 삭제할 Thread 명
   삭제할 Thread 명 표시
3. 확인 [Enter] 버튼
	해당 Thread 삭제하기
4. 취소[ESc] 버튼
   Thread 삭제하지 않고 팝업 닫기
5. X
	Thread 삭제하지 않고 팝업 닫기
## Thread 목록
1. Threads 탭 옆 숫자
   전체 Thread 중 새로운 AI 응답이 있으나 읽지 않은 개수 표시
2. Thread 생성 +
	클릭시 새 Thread 생성
	3초 이내 중복 생성 시 경고 표시
3. 각 Thread 우측 N
	해당 Thread 내 읽지 않은 응답 존재시 표시
4. Thread 검색 기능
	대소문자 구분 없이 기본 Thread 명 또는 지정한 Thread명 검색
	돋보기 아이콘 또는  Enter 키로 검색 실행
	검색 후 결과에 맞춰 Thread 리스트 재정렬
	X 아이콘 클릭 시 검색 초기화
5. Thread 우클릭
	팝업 창으로 Thread 열기
	팝업으로 열린 Thread는 주황색 텍스트로 표시
6.  Thread 좌클릭
   우측 패널에서 Thread 열기
   열린 Thread는 진한 색상으로 표시

# Thread 상세 화면
## Header 영역
1. 현재 선택된 Thread 제목
   우클릭 시Thread ID 복사
   복사시 토스트 메시지 나타났다 사라짐
   기본(default) Thread 제목은 'New Chat'
   기본 Thread 제목 상태('New Chat')에서 처음 질의에 대한 AI 답변 응답 시 해당 응답 요약 내용으로 Thread 명 자동 변경됨
2. Thread 명 변경 버튼
   클릭 시 Thread 명 편집 모드(3) 로 변경
3. Thread명 편집 모드
	Thread 제목이 텍스트 입력란으로 변경
	체크 아이콘 클릭 또는 Enter 키로 텍스트 입력란에 적용된 Thread 명 으로 변경
	X아이콘 클릭 시Thread명 편집 모드 해제
4. 토스트 메시지
5. Thread 고정
	thread 는 사용자가 설정한  Thread 삭제 주기에 따라 해당 주기가 지나면 삭제된다(Thread 생성일자 기준)
	노란 핀 아이콘 : 사용자가 설정한 Thread 삭제 주기의 삭제 대상에서 제외되어 설정된 주기가 지나도 해당 Thread가 자동 삭제 되지 않게 고정
	검정 핀 아이콘 : 사용자가 설정한 Thread 삭제 주기의 삭제 대상으로 설정된 주기가 지나면 자동으로 삭제
## 질의 입력 영역
1. 최근 4개 Thread 리스트 및 '새 스레드' 버튼 출력
	+버튼 클릭 시 최근 5개 Thread 리스트 및 '새 스레드' 버튼 춮력
	최근 5개 Thread 중 하나의 Thread 클릭 시 해당 Thread로 이동
	'새 스레드' 클릭 시 새로운 Thread 생성 후 생성된 Thread로 이동
2. 질의문 입력란
	질의문 입력 후 Enter 키 또는 전송 버튼(3) 클릭 시 질의 요청
	질의 요청 후 AI응답이 올 때 까지 '질의 금지 모드'로 변경(AI 응답이 오기 전까지는 같은 Thread에서 추가 질의 불가)
	'질의 금지모드'(입력불가상태)
		전송 버튼이 정지모양 아이콘으로 변경되면서 비활성화
3. 파일 업로드
   질의할 내용의 파일을 업로드하여 AI에게 관련 질의 요청 가능
   버튼 클릭 시 업로드할 파일 선택 가능
   파일 선택 시 4에 업로드된 파일명 표시
   최대 1개 파일까지 업로드 가능
4. #명령어
   #명령어를 통해 현재 참여중인 Channel과 DM목록 출력
   방향키 또는 마우스를 통해 선택할 방 이동
   Enter 또는 마우스 클릭을 통해 방선택
   방 선택 시 해당 방 제목과 Channel Id를 입력란에 출력
   출력 예시
5. @ 명령어
   채널 선택 후 @명령어 입력 시 선택된 채널의 참여자 출력
   방향키 또는 마우스를 통해 선택할 사용자 이동
   Enter 또는 마우스 클릭을 통해 사용자 선택
   사용자 선택 시 해당 사용자 이름과 사번을 입력란에 출력
   출력 예시

## AI 비서 가이드 창
1. 가이드 창  버튼
   버튼 클릭 시 가이드창 열리거나 닫힘
2. AI 챗봇 서비스 목록 확장 및 축소
   버튼 클릭 시 AI 챗봇 서비스 목록 확장 축소 가능
   확장 시 요청 가능한 AI 챗봇 서비스 목록 출력
3. AI가이드 버튼
   버튼 클릭 시 새로운 Thread 생성 후 생성된 스레드에 각 버튼에 해당되는 AI 가이드 응답 메시지 반환
4. CUBE AI 사용 가이드 지식블로그
   이번 프로젝트를 통해 추가된 새로운 기능을 설명하는 지식블로그 글 url로 이동
5. AI 가이드 버튼 별 응답 메시지 예시
# 신규 Rich 메시지 기능
## 공통 기능
1. 리치 메시지 우클릭
   메시지 우클릭 시 특정된 리치 메시지 전문을 복사
   복사시 2 토스트 메시지 나타났다 사라짐
2. 출처 버튼
   리치 메시지의 출처가 iflow·Confluence·Cube 채널로 이동
3. X 버튼
	출처 목록 팝업 닫기
4. 지식블로그 보내기 버튼
	지식블로그로 편집 가능한 리치메시지인 경우 해당버튼 생성
	리치 메시지 전문을 편집 및 지식블로그에 발행할 수 있는 지식 블로그편집모드 페이지로 이동
	리치메시지 전문을 편집 및 발행할 수 있게 해줌.
	리치 메시지 앞 10글자를 제목으로 적용
	예시화면
5. 마크다운 뷰어 버튼
   마크다운 문법이 적용된 리치메시지인 경우 마크다운 뷰어 버튼 생성
   클릭 시 리치 메시지 전문을 마크다운 문법이 적용된 화면을 볼 수 잇는 마크다운 뷰어 팝업 출력
6. X 버튼
   마크다운 뷰어 팝업 닫기

## CH/DM 화면
1. 신규 리치 메시지인 경우
   상단 안내문구 추가 " CUBE AI에서 작성돈 메시지 입니다."
2. 삭제 버튼
	클릭 시 해당 리치 메시지 삭제
## Thread 상세 화면
1. 답변품질 불만족 버튼
   사용자가 AI응답 의 답변품질이 불만족스러울 경우 체크
2. 응답속도 불만족 버튼
   사용자가 AI응답의 응답속도가 불만족스러울 경우 체크
3. 복사버튼
   리치 메시지 전문을 CH/DM방으로 복사하는 버튼
   버튼 클릭 시 채널간 메시지 복사 팝업이 열림
4. 저장버튼
   선택한 CH/CM 방에 리치 메시지 복사
5. 취소 버튼
   채널간 메시지 복사 팝업 닫기
6. X 버튼
   채널간 메시지 복사 팝업 닫기
# 단축키
## Ctrl + E 기능 상세
1. 우측 패널이  Thread 화면이 아닌경우
	우측 패널이 Thread 상태가 아닌 상태에서 Ctrl + E 키 입력시 새로운 Thread가 생성되고 우측패널에 해당 Thread가 열링
2. 우측 패널에 Thread 화면이 열린 상태에서 닫혀있는 경우
   닫혀 있던 우측 패널 열기
# AI 설정
## AI 검색화면 템플릿 설정
1. 개인 환경설정 버튼
   개인 환경 설정 버튼을 통해 AI 설정 화면으로 이동
2. AI  검색 화면 템플릿 설정
   6가지 템플릿 중 원하는 템플릿 선택
## AI 검색 화면 기간
1. AI 검색화면 기간
   AI Thread 에서 요청 시 Default, 지난주, 지난달 검색 범위 설정
2. 기간 선택 바
   마우스 좌우 스크롤 및 클릭을 통해 기간 선택
3. 선택된 기간 표시
   왼쪽 선택 바 에서 선택된 일자 표시
   방향키 위, 아래 키 입력 시 1일씩 증감 및 ㄴ 왼쪽 선택 바 이동
4. 저장 버튼
   선택된 AI 검색화면 템플릿과 AI 검색 화면 기간 저장
5. 취소버튼
   개인 환경설정 창 닫기
6. X 버튼
   개인 환경설정 창 닫기
