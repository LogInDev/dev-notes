📌 THREAD 관리 기능

⸻

1. Thread 생성
	1.	Thread 생성 버튼
	•	클릭 시 AI Thread 생성
	•	Thread 목록에 New Chat 추가
	•	생성된 Thread가 우측 패널에 열림
	•	3초 이내 중복 생성 시 경고 표시
	2.	Thread 생성(+) 버튼
	•	클릭 시 새 Thread 생성
	•	3초 이내 중복 생성 시 경고 표시
	3.	CH/DM 입력란: /AI, /먀 명령어
	•	/AI 또는 /먀 입력 후 질의 입력 → Enter 시 새 Thread 생성 및 질의 요청
	•	/AI는 대소문자 구분 없음
	4.	CH/DM 입력란: @@ 명령어
	•	@@ 입력 시 새 Thread 생성 및 우측 패널 자동 열림
	•	@@질의 입력 후 Enter 시 새 Thread로 질의 요청
	•	입력 예시 추가 가능

⸻

2. Thread 삭제
	1.	Thread 삭제 아이콘(X)
	•	마우스 오버 시 표시
	•	클릭 시 삭제 확인 팝업 표시
	2.	삭제 팝업 – Thread 명 표시
	3.	확인(Enter)
	•	해당 Thread 삭제
	4.	취소(Esc)
	•	삭제 취소 및 팝업 닫기
	5.	팝업 X 버튼
	•	삭제 취소 및 팝업 닫기

⸻

3. Thread 목록
	1.	Threads 탭 숫자 표시
	•	전체 Thread 중 읽지 않은 AI 응답 수 표시
	2.	Thread 생성(+) 버튼
	•	새 Thread 생성
	•	3초 이내 중복 생성 시 경고
	3.	각 Thread 우측 ‘N’ 표시
	•	해당 Thread 내 읽지 않은 응답 존재 시 표시
	4.	Thread 검색 기능
	•	기본 Thread명/수정된 Thread명 검색 가능 (대소문자 구분 없음)
	•	돋보기 아이콘 또는 Enter로 검색
	•	검색 결과 기준으로 목록 재정렬
	•	X 아이콘: 검색 초기화
	5.	Thread 우클릭
	•	팝업 창으로 Thread 열기
	•	팝업에선 Thread 명이 주황색으로 표시
	6.	Thread 좌클릭
	•	우측 패널에서 Thread 열기
	•	열려 있는 Thread는 진한 색상으로 표시

⸻

📌 THREAD 상세 화면

⸻

1. Header 영역
	1.	선택된 Thread 제목
	•	우클릭 시 Thread ID 복사 (토스트 메시지 표시)
	•	기본 제목: New Chat
	•	기본 제목 상태에서 첫 AI 응답이 생성되면 응답 요약으로 자동 변경
	2.	Thread 명 수정 버튼(연필 아이콘)
	•	클릭 시 편집 모드로 전환
	3.	Thread 명 편집 모드
	•	제목이 입력란으로 변경
	•	체크 아이콘 / Enter: 변경 저장
	•	X 아이콘: 편집 취소
	4.	Thread 고정(Pin)
	•	Thread는 ‘사용자 설정 삭제 주기’에 따라 자동 삭제
	•	노란 핀: 삭제 대상에서 제외(고정)
	•	검정 핀: 삭제 대상(주기 경과 시 자동 삭제)

⸻

2. 질의 입력 영역
	1.	최근 Thread 목록 / 새 Thread
	•	‘+’ 클릭 시 최근 5개 Thread + ‘새 스레드’ 표시
	•	Thread 클릭 시 이동
	•	‘새 스레드’ 클릭 시 새 Thread 생성 후 이동
	2.	질의 입력란
	•	Enter 또는 전송 버튼 클릭 시 질의 요청
	•	AI 응답 도착 전까지 질의 금지 모드
	•	전송 버튼이 비활성화 및 정지 아이콘으로 변경
	3.	파일 업로드
	•	파일 기반 질의 가능
	•	파일 1개 업로드 가능
	•	선택된 파일명은 입력란 하단에 표시
	4.	# 명령어 – CH/DM 목록 호출
	•	참여 중인 채널/DM 목록 표시
	•	방향키/마우스로 선택 후 Enter 클릭
	•	선택 시 해당 방 제목과 채널 ID 자동 입력
	5.	@ 명령어 – 사용자 목록 호출
	•	채널 선택 후 @ 입력 시 사용자 목록 표시
	•	선택 시 사용자 이름 + 사번 자동 입력

⸻

📌 AI 비서 가이드 창
	1.	가이드 창 버튼
	•	클릭 시 열기/닫기
	2.	AI 서비스 목록 확장/축소
	•	서비스 목록 표시 및 숨김
	3.	AI 가이드 버튼
	•	클릭 시 새 Thread 생성
	•	가이드 유형에 맞는 AI 응답 자동 출력
	4.	CUBE AI 사용 가이드(지식블로그)
	•	기능 설명 블로그 URL 이동
	5.	AI 가이드 버튼별 응답 예시 제공

📌 신규 Rich 메시지 기능

⸻

1. 공통 기능
	1.	리치 메시지 우클릭
	•	전문 복사
	•	토스트 메시지 표시
	2.	출처 버튼
	•	iflow / Confluence / Cube 채널 등 출처 페이지로 이동
	•	출처 목록 팝업 제공
	3.	출처 팝업 X
	•	팝업 닫기
	4.	지식블로그 보내기 버튼
	•	편집 가능한 메시지인 경우 활성화
	•	블로그 편집 화면으로 이동
	•	메시지 첫 10글자를 제목으로 자동 적용
	5.	마크다운 뷰어 버튼
	•	마크다운 타입 메시지인 경우 활성화
	•	클릭 시 마크다운 뷰어 팝업 표시
	6.	마크다운 팝업 X
	•	팝업 닫기

⸻

2. CH/DM 화면
	1.	신규 리치 메시지의 경우 상단 안내 문구 표시
→ “CUBE AI에서 작성된 메시지입니다.”
	2.	삭제 버튼
	•	클릭 시 메시지 삭제

⸻

3. Thread 상세 화면
	1.	답변 품질 불만족 버튼
	2.	응답 속도 불만족 버튼
	3.	복사 버튼
	•	리치 메시지를 CH/DM으로 복사
	•	채널 선택 팝업 열림
	4.	저장 버튼
	•	선택한 CH/DM으로 메시지 복사
	5.	취소 / X 버튼
	•	복사 팝업 닫기

⸻

📌 단축키

Ctrl + E

	•	새 Thread 생성 및 우측 패널 열림
	2.	우측 패널이 닫힌 상태일 때
	•	닫힌 Thread 패널 열기

📌 AI 설정

⸻

1. AI 검색 화면 템플릿
	1.	개인 환경설정 진입
	2.	6종 템플릿 중 선택 가능

⸻

2. AI 검색 기간 설정
	1.	Default / 지난주 / 지난달 범위 설정
	2.	기간 슬라이더로 날짜 선택(스크롤/클릭)
	3.	선택된 기간은 좌측 바에 표시
	4.	방향키 ↑↓로 1일 단위 조정 가능
	5.	저장 버튼: 템플릿 + 기간 저장
	6.	취소/X 버튼: 창 닫기
